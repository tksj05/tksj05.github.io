一道入门题的输入格式一起的错误

https://vijos.org/training/58b3aadad3d8a17783ca1aa9

描述

输入两个自然数，输出他们的和
格式

输入格式

两个自然数x和y 0 <= x, y <= 327670<=x,y<=32767
输出格式

一个数，即x和y的和
样例1

样例输入1

123 500
样例输出1

623

一开始格式输入理解有误，实际应为字符串

s = raw_input()
s = s.split()
a = int(s[0])
b = int(s[1])
print a+b
